Um, fixando aspectos
1, acrescentou a função das preferências, muitos aspectos podem agora ser configurados
Em segundo lugar, as letras
1, aumentando a autonomia da pesquisa letras, o usuário pode selecionar uma palavra-chave de pesquisa personalizada, você também pode definir se e letras do arquivo associado com a associação aqui não é para escrever letras arquivos de música, mas para estabelecer um programa em Vinculação, para não pesquisar as letras cada vez que a música é tocada, mas ler diretamente o arquivo de letras associado.
2, aumentando as letras especificadas diretório de pesquisa, especifique o diretório para salvar a função de letras. Por motivos de desempenho, não recursiva letras de busca diretório para baixo, e que apenas as configurações de pesquisa de diretório.
3, conjunto mais rico de letras, um monte de lugares costumavam ser fixo, e agora você pode mudar, e aumentando o retrato de função letras OK exibição Kara, também suporta opção de fonte anti-aliasing é exibida.
4, suporte para letras de exibição desvanece-se, suporta uma variedade de alinhamento.
5. Suporte a exibição transparente da janela de letras sob o windows.Em linux, esta opção será desativada:
Em terceiro lugar, o rótulo
1, já suporta ID3v1, ID3v2, APEv2, leitura tag Vorbis e escrita, e pode definir a codificação para leitura e escrita tags. Ilegível, a fim de evitar a situação ao ler o rótulo. Configuração O padrão de codificação programa O GBK, se for usuário UTF-8, pode ser modificado nas preferências.
Em quarto lugar, outros aspectos
1, você pode escolher o dispositivo de áudio, que até certo ponto pode escolher várias configurações para comparar o efeito.Minimize o jogador sem som.
2. Adicione títulos de músicas de rolagem na barra de tarefas.
3, suporte para a criação de um servidor proxy para se conectar à rede.
4, suporte para a lista de reprodução para fazer algumas configurações detalhadas, porque a cor padrão do programa não necessariamente atende a estética de todos.
5, diretório de configuração agora mudou {user.home} /. YOYOPlayer /, o diretório padrão é a letra {user.home} /. YOYOPlayer / Letras /
